   ║  🚀 GET STARTED IN 3 STEPS                                   ║
   ╚═══════════════════════════════════════════════════════════════╝

   STEP 1: Open Command Prompt in CommAi folder
   ────────────────────────────────────────────────────────────────
   
   Right-click on CommAi folder → "Open in Terminal"
   OR
   Press Win+R → type "cmd" → cd to CommAi folder


   STEP 2: Install Dependencies (First Time Only)
   ────────────────────────────────────────────────────────────────
   
   Run these commands:
   
   pip install -r requirements.txt
   python -m textblob.download_corpora
   
   ⏱️ Takes about 2-3 minutes


   STEP 3: Launch the Application
   ────────────────────────────────────────────────────────────────
   
   Double-click: start.bat
   
   OR run in terminal:
   
   start.bat
   
   🌐 Browser will open automatically at http://localhost:8000

   ╔═══════════════════════════════════════════════════════════════╗
   ║  🔧 OPTIONAL CONFIGURATION                                   ║
   ╚═══════════════════════════════════════════════════════════════╝

   📧 Real Email OTP (Optional)
   ────────────────────────────────────────────────────────────────
   Edit start.bat and add:
   
   set GMAIL_USER=your-email@gmail.com
   set GMAIL_APP_PASSWORD=your-16-digit-password
   
   Without this: OTP prints in terminal (works fine!)


   🤖 AI Chatbot API (Optional)
   ────────────────────────────────────────────────────────────────
   Edit start.bat and add:
   
   set ANTHROPIC_API_KEY=your-key
   OR
   set OPENAI_API_KEY=your-key
   
   Without this: Built-in chatbot works great!
