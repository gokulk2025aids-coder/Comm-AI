⚙️ CONFIGURATION OPTIONS
═══════════════════════════════════════════════════════════════════════

OPTIONAL - Real Email OTP:
Edit start.bat and add:
  set GMAIL_USER=your-email@gmail.com
  set GMAIL_APP_PASSWORD=your-16-digit-password

OPTIONAL - AI Chatbot API:
Edit start.bat and add:
  set ANTHROPIC_API_KEY=your-key
  OR
  set OPENAI_API_KEY=your-key

Note: Both are optional! System works great without them.

⚠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Problem: Module not found
Solution: pip install -r requirements.txt

Problem: Port already in use
Solution: Edit start.bat, change port to 8001

Problem: OTP not received
Solution: Check terminal window (dev mode prints OTP)

Problem: TextBlob errors
Solution: python -m textblob.download_corpora

Problem: Database errors
Solution: Delete commai.db file and restart

Problem: Browser doesn't open
Solution: Manually open http://localhost:8000
