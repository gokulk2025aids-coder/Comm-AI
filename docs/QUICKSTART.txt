📋 QUICK START GUIDE
═══════════════════════════════════════════════════════════════

✅ STEP 1: Install Python Dependencies
───────────────────────────────────────────────────────────────
Open Command Prompt in this folder and run:

    pip install -r requirements.txt
    python -m textblob.download_corpora


✅ STEP 2: Run the Application
───────────────────────────────────────────────────────────────
Windows:
    Double-click: start.bat

Mac/Linux:
    cd backend
    python -m uvicorn main:app --reload --port 8000


✅ STEP 3: Open in Browser
───────────────────────────────────────────────────────────────
    http://localhost:8000


✅ STEP 4: Login
───────────────────────────────────────────────────────────────
Choose one of three methods:
    • OTP Login (email-based, OTP prints in terminal)
    • Password Login (if you have an account)
    • Sign Up (create new account)

⚙️ OPTIONAL CONFIGURATION
═══════════════════════════════════════════════════════════════

📧 Real Email OTP (Optional):
Edit start.bat and uncomment:
    set GMAIL_USER=your-email@gmail.com
    set GMAIL_APP_PASSWORD=your-16-digit-app-password

🤖 AI Chatbot API (Optional):
Edit start.bat and add:
    set ANTHROPIC_API_KEY=your-key
    OR
    set OPENAI_API_KEY=your-key

Note: Built-in chatbot works great without any API key!

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ "Module not found"
   → Run: pip install -r requirements.txt

❌ "Port already in use"
   → Change port in start.bat to 8001

❌ "OTP not received"
   → Check terminal window (OTP prints there in dev mode)

❌ Database errors
   → Delete commai.db file and restart
